# 💰 Java Expense Tracker

A robust command-line interface (CLI) application built to efficiently manage and track daily financial transactions. Designed with a focus on clean Object-Oriented Programming (OOP) architecture and modularity.

## ✨ Features
* **Interactive Dashboard:** Easy-to-navigate CLI menu for seamless user experience.
* **Transaction Management:** Add, categorize, and view daily expenses.
* **Scalable Architecture:** Codebase structured for easy integration with persistent database storage.

## 🛠️ Tech Stack
* **Language:** Java 21
* **Build Tool:** Maven
* **Core Concepts:** OOP, Data Structures, File I/O (Planned)
